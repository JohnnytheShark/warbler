# Contributing to Warbler 🐦

First off, thank you for considering contributing to Warbler! It's people like you that make the local AI ecosystem so great.

## How Can I Help?

### Reporting Bugs
-   Check if the issue has already been reported.
-   Use a clear and descriptive title.
-   Describe the exact steps which reproduce the problem.

### Suggesting Enhancements
-   Explain the behavior you would like to see and why.
-   Provide specific examples of how the feature would work.

### Pull Requests
1.  Fork the repo and create your branch from `main`.
2.  Ensue your code follows the existing style.
3.  Test your changes thoroughly.
4.  Open a Pull Request with a clear description of the problem and your solution.

## Development Setup

See the [Setup section in the README](./README.md#setup) to get your local environment ready.

## Code of Conduct

Help us keep Warbler a welcoming and inclusive project by being kind and respectful to all contributors.
